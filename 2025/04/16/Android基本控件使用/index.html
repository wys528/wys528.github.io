EditText

1. username!= null

此条件用于检查 username 这个字符串对象是否为 null。在 Java 中，null 代表对象引用不指向任何实际的对象。要是 username 为 null，那么调用 username.length() 方法时就会抛出 NullPointerException 异常。所以，先检查 username 是否为 null 是为了避免出现这种异常。

2. username.length() != 0

该条件用于检查 username 字符串的长度是否为 0。即使 username 不是 null，它也可能是一个空字符串（例如 ""）。当字符串为空时，其长度为 0。这个条件可以确保输入的字符串不为空。
